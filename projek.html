Pengalaman Proyek

  • Simulasi Jaringan OSPF & BGP (2025): Merancang dan mengonfigurasi topologi jaringan yang menghubungkan jaringan privat (OSPF) dan publik (BGP) menggunakan beberapa router dan server. Proyek ini memperkuat pemahaman saya tentang routing dinamis dan manajemen trafik antar jaringan. Saya juga menguji konektivitas dan keamanan antar node menggunakan skenario realistis berbasis Cisco Packet Tracer.
  • Website Portofolio UKM POWERS (2025): Membangun website portofolio responsif menggunakan HTML dan CSS untuk menampilkan karya dan informasi UKM POWERS. Saya berkolaborasi dengan tim UKM untuk memahami kebutuhan konten dan menerjemahkannya ke dalam struktur web yang efektif. Proyek ini mengasah kemampuan saya dalam desain antarmuka, pengujian lintas browser, dan pengembangan web yang berorientasi pada pengguna.
